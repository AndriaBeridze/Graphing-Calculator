# Graphing Calculator

## Overview

Welcome to the Graphing Calculator project, developed by AndriaBeridze. This calculator allows differentiation and integration of polynomial functions, aiding in mathematical analysis.

## Features

- **Polynomial Function Support:** The calculator focuses on polynomial functions, offering quick derivatives and integrals.

- **Differentiation:** Compute derivatives of polynomial functions for slope and rate of change analysis.

- **Integration:** Obtain integrals of polynomial functions to analyze accumulated quantities and areas under curves.

- **User-Friendly Interface:** Input polynomial functions, choose differentiation or integration, and receive instant results.

## Getting Started

To use the Graphing Calculator:

1. **Installation:** Run the calculator from a Python-compatible IDE or terminal.

2. **Input:** Input your polynomial function as guided.

3. **Operation:** Select differentiation or integration.

4. **Results:** View calculated results and graphical representations.
